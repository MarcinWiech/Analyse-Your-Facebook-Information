# Analyse-Your-Facebook-Information

### Prerequisites

Run the following commands in your project directory.

```bash
python3 -m virtualenv .
. bin/activate
pip3 install reportlab
pacman -S tk
pip3 install appjar
```
### To run

```bash
./Facebook_Analyzer.sh
```
